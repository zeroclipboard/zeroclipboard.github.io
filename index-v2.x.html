ZeroClipboard v2.x

The ZeroClipboard library provides an easy way to copy text to the clipboard using an invisible Adobe Flash movie and a JavaScript interface. The "Zero" signifies that the library is invisible and the user interface is left entirely up to you.

WARNING!

This page is for the v2.x version of ZeroClipboard, which is currently still in beta. For the stable v1.x, please see the v1.x demo page.

Testing this page locally

To test this page locally, checkout the gh-pages branch and run make to open localhost:3000, then navigate to index-v2.x.html.
